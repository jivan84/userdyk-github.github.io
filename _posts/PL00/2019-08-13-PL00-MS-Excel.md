## **VBA basic**
### ***Tutorials***
#### MsgBox
```vba
Sub procedure()
    MsgBox "Hello World!"
End Sub
```

#### Sub Procedure
```vba
Sub procedure()
    Range("C1") = Now()
End Sub
```
`Private`
```vba
Private Sub procedure()
    Range("C1") = Now()
End Sub
```
`Public`
```vba
Public Sub procedure()
    Range("C1") = Now()
End Sub
```

#### Function Procedure
```vba
Function procedure(x, y)
     procedure = x + y
End Function
```

#### Calling Sub Procedure
```vba
Sub procedure1(a)
    MsgBox a
End Sub

Sub procedure2()
    Call procedure1("ABC")
End Sub
```

### ***Decision Structures - IF and Select Case***
#### IF ... Then Statement
```vba
Sub procedure()
    Ago = 20
    If Ago >= 18 Then
        Status = "Adult"
        Vote = "Yes"
    End If
    MsgBox (Status & vbCrLf & Vote)
End Sub
```

<br><br><br>

---

### ***Loop Structures***
#### For ... Next
```vba
Sub procedure()
    For i = 1 To 10
        Cells(i, 1) = i
    Next i
End Sub
```

#### For ... Next Loop With Step
```vba
Sub procedure()
    For i = 1 To 10 Step 2
        Cells(i, 1) = i
    Next i
End Sub
```

#### Do While ... Loop
```vba
Sub procedure()
   i = 1
    Do While i <= 10
        Cells(i, 1) = i
        i = i + 1
    Loop
End Sub
```

#### Do Until ... Loop
```vba
Sub procedure()
    i = 1
    Do Until i = 11
        Cells(i, 1) = i
        i = i + 1
    Loop
End Sub
```

#### Do ... Loop While
```vba
Sub procedure()
   i = 1
    Do
        Cells(i, 1) = i
        i = i + 1
    Loop While i < 11
End Sub
```

#### Do ... Loop Until
```vba
Sub procedure()
   i = 1
    Do
        Cells(i, 1) = i
        i = i + 1
    Loop Until i = 11
End Sub
```
